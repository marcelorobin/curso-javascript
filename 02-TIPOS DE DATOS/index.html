Tipos de Datos

En JavaScript existen los siguientes tipos de datos:

  • Números: Pueden ser enteros o decimales.
  • Cadenas de Texto: Representadas entre comillas simples o dobles.
  • Booleanos: Pueden ser valores "true" o "false".
  • Undefined: Valor que se asigna a una variable sin un valor específico.
  • Null: Valor que representa la ausencia de valor.
  • Objetos: Estructuras de datos más complejas que contienen propiedades y métodos.
  • Arrays: Listas ordenadas de valores.
  • Funciones: Bloques de código reutilizables.
  • Clases: Son un tipo de funciones.
  • Símbolos: Valores únicos que se utilizan como identificadores para propiedades de objetos.

Todos estos tipos de datos se pueden asignar a variables y manipular a través de operadores y funciones en JavaScript.
